# Scientific Calculator with History using Python

This project is a Scientific Calculator developed in Python that supports both basic and advanced mathematical operations while maintaining a history of calculations. The application enables users to perform complex computations and review previous results, making calculations more efficient and user-friendly.

## Features

* Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* Scientific functions (Square Root, Power, Logarithm, Trigonometric Functions)
* Calculation history tracking
* View and manage previous calculations
* Input validation and error handling
* User-friendly interface
* Fast and accurate mathematical computations

## Technologies Used

* Python
* Math Module
* Tkinter (for GUI, if applicable)

## Objectives

The goal of this project is to demonstrate Python programming, mathematical computation, data handling, and user interface development by building a functional scientific calculator with a history management feature.

## Key Concepts Implemented

* Functions and Modular Programming
* Exception Handling
* Mathematical Computations
* Data Storage and Retrieval
* GUI Development (if applicable)
* User Interaction Design

## How to Run

1. Clone the repository:
   git clone <repository-url>

2. Navigate to the project directory.

3. Run the application:
   python calculator.py
